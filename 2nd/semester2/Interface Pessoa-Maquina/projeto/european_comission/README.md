## Como correr o projeto

### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado

### Passos
```bash
# Entrar na pasta
cd european_comission

# Instalar as dependências
npm install

# Arrancar o projeto
npm run dev

# Formar a dv
npm run build:db

# Correr o json server
npm run api
```

Abrir o browser em no local host que aparece
